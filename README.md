## 🚀 Key Projects

### 🔹 Datadog Monitoring Infrastructure (LSEG)
> **Stack:** AWS · Terraform · GitLab CI/CD · HashiCorp Vault · Lambda · CloudFormation · Secrets Manager

- Built a modular Terraform repo managing three services: **Datadog** (Secrets Manager integration), **Datadog Forwarder** (via CloudFormation), and **Lambda Datadog** (Python Lambda with Datadog layers)
- Designed a GitLab CI pipeline with **manual-trigger stages**: `test-aws-access → terraform-init → terraform-plan → terraform-apply → destroy`
- Integrated **HashiCorp Vault** for dynamic AWS credential injection — zero static credentials in pipelines
- Managed **remote Terraform state** via S3 + DynamoDB locking across `dev / qa / pe / prod` environments in three AWS regions (`us-east-1`, `ap-southeast-1`, `eu-west-1`)

---

### 🔹 Wealthinbound .NET 8 API — DevSecOps Pipeline (LSEG)
> **Stack:** GitLab CI/CD · Docker · ECR · ECS · Semgrep · SonarQube · Trivy · Vault · BAMS

- Built a fully automated GitLab CI pipeline triggered on `main` branch with **immutable image tags** (`branch + commit SHA`)
- **Stages:** `scan (Semgrep + SonarQube)` → `build-QA (Docker + ECR push us-east-1)` → `retag (copy to ap-southeast-1)` → `Trivy scan` → `deploy-QA (ECS update both regions)` → `BAMS registry push`
- Multi-region image promotion with zero manual intervention; Vault handles all secret injection
- Enforced security scanning gates — pipeline fails on critical CVEs before any deployment

---

### 🔹 API Gateway Deployment Automation (LSEG)
> **Stack:** Python · OAuth2 · REST APIs · LSEG RDP API Gateway

- Reduced a **2-hour manual deployment process to ~15 minutes** using a Python automation script
- Developers supply a ZIP (Swagger YAML + manifest + env configs); script handles the rest
- Implemented **OAuth2 token refresh logic** handling sub-10-minute token expiry transparently
- Automated `cutover` calls across `dev / QA / prod` via `/admin/v1/api-deploy/cutover` endpoint

---

### 🔹 Wealth Advisor Platform — Global AWS Infrastructure (LSEG)
> **Stack:** AWS · Aurora PostgreSQL · DynamoDB · ElastiCache Redis · PrivateLink · Global Accelerator · GuardDuty · CrowdStrike · Terraform

- Architected a **global wealth management platform** across multiple AWS regions with latency-based routing via **AWS Global Accelerator**
- Integrated **LSEG's RDP API Gateway with AWS PrivateLink** for secure, private connectivity
- **Aurora PostgreSQL** with multi-region replication: RPO near-zero, RTO 8–24 hrs; all data encrypted at rest (KMS)
- Security stack: TLS 1.2, AAA authentication, RBAC, Secrets Manager, **GuardDuty + CrowdStrike** for runtime threat detection
- **GDPR-compliant** data ingestion: CSV (small firms), API/SFTP (large firms), ETL with virus scanning + S3 staging
- Cloud-agnostic design prepared for future Azure migration
